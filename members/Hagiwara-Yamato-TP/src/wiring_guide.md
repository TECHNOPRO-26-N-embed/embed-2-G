# ELEGOO UNO R3 ピン接続ガイド（12ピン直結型 4桁7セグ用）

## 概要

このガイドは、12ピン直結型の Four Digital Seven Segment Display を使う場合の配線方法です。
TM1637用（4ピン: CLK/DIO/VCC/GND）とは別方式です。

注意: 12ピン直結型では `TM1637Display` ライブラリは使いません。現在の `slot_machine.ino` はTM1637前提のため、そのままでは動作しません。

---

## 1. Four Digital Seven Segment Display（12ピン直結型）

12ピン直結型では、表示セグメント8本（A,B,C,D,E,F,G,DP）と桁選択4本（DIG1-4 / COM1-4）をUNOへ直接つなぎます。

### 先に準備するもの

- 220Ω抵抗 x 8（A,B,C,D,E,F,G,DP 各ラインに1本）
- NPNトランジスタ x 4（2N2222など。桁選択側の電流確保用）
- ベース抵抗 1kΩ x 4（トランジスタのベース用）

### UNOピン割り当て例（競合しない構成）

| 表示側信号 | UNO R3ピン |
|:----------:|:----------:|
| A          | 2          |
| B          | 9          |
| C          | 10         |
| D          | 11         |
| E          | 12         |
| F          | 13         |
| G          | A1         |
| DP         | A2         |

| 桁選択信号 | UNO R3ピン |
|:----------:|:----------:|
| DIG1       | A3         |
| DIG2       | A4         |
| DIG3       | A5         |
| DIG4       | 0          |

### 接続のポイント

- Display側に `A/B/C/D/E/F/G/DP` と `DIG1-4`（または `COM1-4`）の印字があるか確認する
- セグメント線（A〜DP）は必ず220Ω抵抗を直列に入れる
- 桁選択線（DIG/COM）はトランジスタ経由で駆動する
- モジュールが共通アノードか共通カソードかで、点灯ロジック（HIGH/LOW）が逆になる

### 注意

- `D0` はUSBシリアル通信用ピンです。書き込み時に不安定なら、`DIG4` を別ピンへ再割り当てしてください。
- 12ピン直結型はTM1637と互換ではないため、`TM1637Display` ライブラリは使えません。

---

## 2. アナログスティック（Y軸のみ使用）

| 機能         | スティック端子 | UNO R3ピン |
|:------------:|:--------------:|:----------:|
| Y軸出力      | VRy            | A0         |
| VCC          | VCC            | 5V         |
| GND          | GND            | GND        |

---

## 3. ストップボタン（3個）

| ボタン       | UNO R3ピン |
|:------------:|:----------:|
| STOP1        | 3          |
| STOP2        | 4          |
| STOP3        | 5          |

※各ボタンの片側はGNDへ、もう片側を上記ピンへ接続（プルアップ入力）

---

## 4. LED（3個、順番点灯用）

| LED番号      | UNO R3ピン |
|:------------:|:----------:|
| LED1         | 6          |
| LED2         | 7          |
| LED3         | 8          |

※各LEDのアノード（長い足）をピン、カソード（短い足）をGNDへ。抵抗（220Ω程度）を直列に入れること。

---

## 5. 電源

- UNO R3の5Vピンから各部品のVCCへ
- UNO R3のGNDピンから各部品のGNDへ

---

## 6. 配線図イメージ

```
[4桁7セグ 12ピン直結型]
	A  -> D2   (220Ω直列)
	B  -> D9   (220Ω直列)
	C  -> D10  (220Ω直列)
	D  -> D11  (220Ω直列)
	E  -> D12  (220Ω直列)
	F  -> D13  (220Ω直列)
	G  -> A1   (220Ω直列)
	DP -> A2   (220Ω直列)
	DIG1 -> A3 (トランジスタ経由)
	DIG2 -> A4 (トランジスタ経由)
	DIG3 -> A5 (トランジスタ経由)
	DIG4 -> D0 (トランジスタ経由)

[スティック]---[A0:VRy] [5V] [GND]
[STOP1]---[3] [GND]
[STOP2]---[4] [GND]
[STOP3]---[5] [GND]
[LED1]---[6]---[抵抗]---[GND]
[LED2]---[7]---[抵抗]---[GND]
[LED3]---[8]---[抵抗]---[GND]
```

---

このガイドは12ピン直結型向けです。ディスプレイ制御コードも直結型用に変更して使ってください。
