## 1. 시스템 전체 구조

### 1-1. 블록 구성

```text
【입력】                 【Arduino UNO R3】            【출력】
PIR 인체감지 센서 ───→┐                         ┌──→ DC 모터(선풍기)
                     │   사람 감지 및 타이머 관리 ├──→ 4Digit 7-Segment Display
                     │   PWM 제어 처리           └──→ LED(동작 상태 표시)
                     └─────────────────────────┘
```

보충 설명:

PIR 인체감지 센서로 사람의 움직임을 감지하고, Arduino UNO R3가 감지 상태를 판정한다. 사람을 감지하면 DC 모터를 PWM 제어로 회전시키고, PWM 값을 4Digit 7-Segment Display에 표시한다. 20초 동안 감지가 없으면 모터를 정지시키고, 7세그먼트 디스플레이에 “OFF”를 표시한다.

---

### 1-2. 상태 전이

```text
[전원 ON / 초기화]
  ↓（초기화 완료）
[대기 중] ──（사람 감지）──→ [동작 중] ──（20초간 미감지）──→ [정지 상태]
   ↑                                                          │
   └──────────────（다시 사람 감지）──────────────────────────┘
```

| 상태명 | 상태 설명 | 출력 상태 |
|:--|:--|:--|
| 대기 중 | PIR 센서 값을 감시하여 사람을 감지 | LED OFF, 7세그 OFF 표시 |
| 동작 중 | 모터를 PWM 제어로 회전시키고 PWM 값을 증가 | LED ON, PWM 값 표시 |
| 정지 상태 | 모터를 정지하고 다시 감지될 때까지 대기 | LED OFF, 7세그 OFF 표시 |

## 3. 하드웨어 설계

### 3-1. 부품 및 핀 연결

| 부품명 | 종류 | Arduino 핀 | 입출력 | 비고 |
|:--|:--|:--|:--|:--|
| PIR 인체감지 센서 | HC-SR501 | D7 | 입력 | 사람 감지 |
| DC 모터 | Fan Blade and DC 3-6V Motor | L293D 연결 | 출력 | PWM 속도 제어 |
| 모터 드라이버 | L293D | D3, D4, D5 | 출력 | 모터 제어 |
| 4Digit 7-Segment | 12핀 타입 | D8, D9, D10 | 출력 | PWM/OFF 표시 |
| LED | 5mm LED | D13 | 출력 | 상태 표시 |

핀 충돌 결과: 충돌 없음

- D0/D1 미사용
- D3/D4/D5는 L293D 전용
- D7은 PIR 센서 전용
- D8/D9/D10은 7세그 전용
- D13은 LED 전용
